🍽 The Hungry Base
A Full-Stack Food Ordering System (MERN Stack)

The Hungry Base is a Zomato-style full-stack food ordering web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). It allows users to browse multiple food categories, add items to their cart, and place orders by providing delivery details.

🚀 Features

🏪 Multiple Food Sections (Starters, Main Course, Desserts, Beverages, etc.)

🛒 Add to Cart with real-time cart management

🔄 Dynamic cart value calculation (auto total update)

👤 User Information Collection (Name, Phone Number, Address)

💾 Order and Cart Data Stored in MongoDB Database

🔐 Secure Backend APIs for Order Processing

📱 Fully Responsive Design (Mobile + Desktop)

⚡ Fast and Smooth User Experience

🧠 How It Works

User browses different food categories.

User adds food items to the cart.

Cart automatically updates quantity and total price.

User enters:

Name

Phone Number

Address

Order details + cart items are stored securely in MongoDB.

Backend APIs handle order submission and data storage.
